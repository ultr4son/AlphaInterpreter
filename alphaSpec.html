Registers

A: Accumulator

Z: Anti Accumulator (-A)

S: Stack, gives 0 if empty.

I: Next available input in numerical form, gives 0 if empty

Commands

a[n/Z/S/I]: Add a number or register to A

b[n/Z/S/I]: Jump a number or register amount of instructions if A > 0

c[n/Z/S/I]: Jump a number or register amount of instructions if A = 0

d[n/Z/S/I]: Jump a number or register amount of instructions if A < 0

e[n/Z/S/I]: Jump a number or register amount of instructions

g[n/Z/S/I]: Assign a number or register to A

o: Output A as an ascii encoded character

n: Output A as a number

p: Push A onto S

l: Pop S onto A

f: Remove all pushed values from S

z: Do nothing

*: Attaches to the instruction behind it. After the instruction executes, the running environment will breakpoint.
